# IP Calc

## Summary

Simple [vuejs](https://vuejs.org/)-powered app for getting info from IPv4 CIDR ([ip](https://www.npmjs.com/package/ip) nodejs package as "backend"). Bootstrapped via `vue create vue-ip-calc`.
